# Fermion

Functions for the study of fermionic systems.

## Functions

### BitstringVectorType

<Function id="BitstringVectorType" signature="template<class BitstringVectorType>auto Qiskit::addon::sqd::bitstrings_to_ci_strings_symmetrize_spin(const BitstringVectorType &bitstrings, std::optional<unsigned int> max_dimension = std::nullopt, std::optional<std::reference_wrapper<const std::vector<internal::HalfSize<typename BitstringVectorType::value_type>>>> include_configurations = std::nullopt) -> std::vector<internal::HalfSize<typename BitstringVectorType::value_type>>">
  Convert bitstrings into CI strings (representations of determinants).

  This function separates each bitstring in bitstring\_matrix in half, combining the right and left halves of all the bitstrings into a single set of unique configurations.

  **Parameters**

  *   **bitstrings** – **\[in]** Population of bitstrings.
  *   **max\_dimension** – **\[in]** Maximum dimension of returned CI strings. If less than the number of CI strings, the list of CI strings will be truncated.
  *   **include\_configurations** – **\[in]** A list of CI strings that will be included in the output, regardless of whether they are contained in `bitstrings`.

  **Template Parameters**

  **BitstringVectorType** – Type of `bitstrings`, compatible with `std::vector<boost::dynamic_bitset<>>`.
</Function>
