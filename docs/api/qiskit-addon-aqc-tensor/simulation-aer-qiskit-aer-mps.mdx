# QiskitAerMPS

  Qiskit Aer representation of a matrix-product state.

  This form of a matrix-product state was introduced in the original time-evolving block decimation (TEBD) paper, [https://arxiv.org/abs/quant-ph/0301063](https://arxiv.org/abs/quant-ph/0301063). See Sec. 7.3.2 of [https://arxiv.org/abs/1008.3477v2](https://arxiv.org/abs/1008.3477v2) for more details on the relationship with other MPS representations.
