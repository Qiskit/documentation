  This is compatible with both [Quimb’s MPS simulator](https://quimb.readthedocs.io/en/latest/tensor-circuit-mps.html), which eagerly contracts gates, as well as [Quimb’s standard method for circuit simulation](https://quimb.readthedocs.io/en/latest/tensor-circuit.html).
