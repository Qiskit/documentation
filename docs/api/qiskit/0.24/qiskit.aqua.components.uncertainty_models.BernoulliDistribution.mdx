  The Univariate Bernoulli Distribution.

  Distribution with only two values (low, high) and the corresponding probabilities represented by a single qubit.

  **Parameters**

  *   **p** (`float`) – Probability
  *   **low** (`float`) – Low value
  *   **high** (`float`) – High value
