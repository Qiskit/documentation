# qiskit.transpiler.FlowController

<Class id="qiskit.transpiler.FlowController" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.17/qiskit/transpiler/runningpassmanager.py" signature="FlowController(passes, options, **partial_controller)" modifiers="class">
  Base class for multiple types of working list.

  This class is a base class for multiple types of working list. When you iterate on it, it returns the next pass to run.
