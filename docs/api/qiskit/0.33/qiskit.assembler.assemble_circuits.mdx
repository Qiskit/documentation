# qiskit.assembler.assemble\_circuits

<Function id="qiskit.assembler.assemble_circuits" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.19/qiskit/assembler/assemble_circuits.py" signature="assemble_circuits(circuits, run_config, qobj_id, qobj_header)">
  Assembles a list of circuits into a qobj that can be run on the backend.

  **Parameters**

  *   **circuits** (`List`\[`QuantumCircuit`]) – circuit(s) to assemble
  *   **run\_config** (`RunConfig`) – configuration of the runtime environment
  *   **qobj\_id** (`int`) – identifier for the generated qobj
  *   **qobj\_header** (`QobjHeader`) – header to pass to the results

  **Return type**

  `QasmQobj`

  **Returns**

  The qobj to be run on the backends
</Function>
