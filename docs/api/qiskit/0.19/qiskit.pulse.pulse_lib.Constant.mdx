# Constant

<Class id="qiskit.pulse.pulse_lib.Constant" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.14/qiskit/pulse/pulse_lib/parametric_pulses.py" signature="Constant(duration, amp, name=None)" modifiers="class">
  A simple constant pulse, with an amplitude value and a duration:

$$
f(x) = amp    ,  0 <= x < duration
f(x) = 0      ,  elsewhere
$$

  Initialize the constant-valued pulse.

  **Parameters**

  *   **duration** (`int`) – Pulse length in terms of the the sampling period dt.
  *   **amp** (`complex`) – The amplitude of the constant square pulse.
  *   **name** (`Optional`\[`str`]) – Display name for this pulse envelope.
