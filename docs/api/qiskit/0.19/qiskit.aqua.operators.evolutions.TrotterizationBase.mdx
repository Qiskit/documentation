# TrotterizationBase

<Class id="qiskit.aqua.operators.evolutions.TrotterizationBase" isDedicatedPage={true} github="https://github.com/qiskit-community/qiskit-aqua/tree/stable/0.7/qiskit/aqua/operators/evolutions/trotterizations/trotterization_base.py" signature="TrotterizationBase(reps=1)" modifiers="class">

  <span id="trotterizationbase-reps" />

  ### reps

  <Attribute id="qiskit.aqua.operators.evolutions.TrotterizationBase.reps" attributeTypeHint="int">

    **Return type**

    `int`
  </Attribute>

    **operator** ([`OperatorBase`](qiskit.aqua.operators.OperatorBase "qiskit.aqua.operators.operator_base.OperatorBase")) – The `SummedOp` to evolve.

    **Return type**

    [`OperatorBase`](qiskit.aqua.operators.OperatorBase "qiskit.aqua.operators.operator_base.OperatorBase")
