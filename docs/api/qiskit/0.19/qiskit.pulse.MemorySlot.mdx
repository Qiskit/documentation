# MemorySlot

<Class id="qiskit.pulse.MemorySlot" github="https://github.com/qiskit/qiskit/tree/stable/0.14/qiskit/pulse/channels.py" signature="MemorySlot(index)">
  Memory slot channels represent classical memory storage.

  Channel class.

  **Parameters**

  **index** (`int`) – Index of channel.

  **Raises**

  [**PulseError**](qiskit.pulse.PulseError "qiskit.pulse.PulseError") – If `index` is not a nonnegative integer.

  ## Attributes

  ### index

  <Attribute id="qiskit.pulse.MemorySlot.index" name="index" signature="int">
    Return the index of this channel. The index is a label for a control signal line typically mapped trivially to a qubit index. For instance, `DriveChannel(0)` labels the signal line driving the qubit labeled with index 0.

    **Return type**

    `int`
  </Attribute>

  ### name

  <Attribute id="qiskit.pulse.MemorySlot.name" name="name" signature="str">
    Return the shorthand alias for this channel, which is based on its type and index.

    **Return type**

    `str`
  </Attribute>

  ### prefix

  <Attribute id="qiskit.pulse.MemorySlot.prefix" signature="= 'm'">
    A shorthand string prefix for characterizing the channel type.
