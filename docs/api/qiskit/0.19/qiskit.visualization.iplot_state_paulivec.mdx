# iplot\_state\_paulivec

<Function id="qiskit.visualization.iplot_state_paulivec" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.14/qiskit/visualization/interactive/iplot_paulivec.py" signature="iplot_state_paulivec(rho, figsize=None, slider=False, show_legend=False)">
  Create a paulivec representation.

  Graphical representation of the input array.

  **Parameters**

  *   **rho** (*array*) – State vector or density matrix.
  *   **figsize** (*tuple*) – Figure size in pixels.
  *   **slider** (*bool*) – activate slider
  *   **show\_legend** (*bool*) – show legend of graph content

  **Example**

  ```python
  from qiskit import QuantumCircuit, BasicAer, execute
  from qiskit.visualization import iplot_state_paulivec
  %matplotlib inline

  qc = QuantumCircuit(2, 2)
  qc.h(0)
  qc.cx(0, 1)
  qc.measure([0, 1], [0, 1])

  backend = BasicAer.get_backend('statevector_simulator')
  job = execute(qc, backend).result()
  iplot_state_paulivec(job.get_statevector(qc))
  ```
</Function>
