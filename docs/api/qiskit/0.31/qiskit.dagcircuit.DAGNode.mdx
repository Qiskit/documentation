# DAGNode

<Class id="qiskit.dagcircuit.DAGNode" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.18/qiskit/dagcircuit/dagnode.py" signature="DAGNode(type=None, op=None, name=None, qargs=None, cargs=None, wire=None, nid=- 1)" modifiers="class">

  Object to represent the information at a node in the DAGCircuit.

  It is used as the return value from \*\_nodes() functions and can be supplied to functions that take a node.
