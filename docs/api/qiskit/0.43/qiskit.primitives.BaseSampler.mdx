# BaseSampler

<Class id="qiskit.primitives.BaseSampler" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.24/qiskit/primitives/base/base_sampler.py" signature="BaseSampler(*, options=None)" modifiers="class">
