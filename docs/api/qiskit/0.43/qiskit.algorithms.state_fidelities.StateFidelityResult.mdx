  ## Attributes

  ### fidelities

  <Attribute id="qiskit.algorithms.state_fidelities.StateFidelityResult.fidelities" attributeTypeHint="Sequence[float]">
    List of truncated fidelity values for each pair of input circuits, ensured to be in \[0,1].
  </Attribute>

  ### raw\_fidelities

  <Attribute id="qiskit.algorithms.state_fidelities.StateFidelityResult.raw_fidelities" attributeTypeHint="Sequence[float]">

  </Attribute>

  ### metadata

  <Attribute id="qiskit.algorithms.state_fidelities.StateFidelityResult.metadata" attributeTypeHint="Sequence[Mapping[str, Any]]">
    Additional information about the fidelity calculation.
