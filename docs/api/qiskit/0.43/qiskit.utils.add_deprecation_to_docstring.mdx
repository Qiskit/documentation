# qiskit.utils.add\_deprecation\_to\_docstring

  **Parameters**

  *   **func** (*Callable*) – The function to modify.
  *   **msg** (*str*) – The full deprecation message.
  *   **since** (*str | None*) – The version the deprecation started at.
