# qiskit.pulse.builder.qubit\_channels

<Function id="qiskit.pulse.builder.qubit_channels" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.17/qiskit/pulse/builder.py" signature="qubit_channels(qubit)">
  Returns the set of channels associated with a qubit.

  Examples:

  ```python
  from qiskit import pulse
  from qiskit.test.mock import FakeOpenPulse2Q

  backend = FakeOpenPulse2Q()

  with pulse.build(backend):
      print(pulse.qubit_channels(0))
  ```

  ```
  {MeasureChannel(0), AcquireChannel(0), ControlChannel(1), ControlChannel(0), DriveChannel(0)}
  ```

  <Admonition title="Note" type="note">
    Requires the active builder context to have a backend set.
  </Admonition>

  <Admonition title="Note" type="note">
    A channel may still be associated with another qubit in this list such as in the case where significant crosstalk exists.
  </Admonition>

  **Return type**

  `Set`\[[`Channel`](qiskit.pulse.channels#channel "qiskit.pulse.channels.Channel")]
</Function>
