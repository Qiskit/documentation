# MCMTVChain

<Class id="qiskit.circuit.library.MCMTVChain" isDedicatedPage={true} github="https://github.com/Qiskit/qiskit/tree/stable/2.2/qiskit/circuit/library/generalized_gates/mcmt.py#L123-L185" signature="qiskit.circuit.library.MCMTVChain(gate, num_ctrl_qubits, num_target_qubits)" modifiers="class">
  Bases: [`MCMT`](qiskit.circuit.library.MCMT "qiskit.circuit.library.generalized_gates.mcmt.MCMT")

  The MCMT implementation using the CCX V-chain.

  This implementation requires ancillas but is decomposed into a much shallower circuit than the default implementation in [`MCMT`](qiskit.circuit.library.MCMT "qiskit.circuit.library.MCMT").

  Expanded circuit:

  ![Diagram illustrating the previously described circuit.](/docs/images/api/qiskit/2.2/qiskit-circuit-library-MCMTVChain-1.avif)

  **Examples**

  ```python
  >>> from qiskit.circuit.library import HGate
  >>> MCMTVChain(HGate(), 3, 2).draw()
  ```
