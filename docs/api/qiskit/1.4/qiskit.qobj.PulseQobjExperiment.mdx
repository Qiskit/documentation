  Each instance of this class is used to represent an individual Pulse experiment as part of a larger Pulse Qobj.

  Instantiate a PulseQobjExperiment.

  <Admonition title="Deprecated since version 1.2" type="danger">

The Qobj class and related functionality are part of the deprecated BackendV1 workflow, and no longer necessary for BackendV2.
