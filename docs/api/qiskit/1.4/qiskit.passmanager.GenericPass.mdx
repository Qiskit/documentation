<Class id="qiskit.passmanager.GenericPass" isDedicatedPage={true} github="https://github.com/Qiskit/qiskit/tree/stable/1.4/qiskit/passmanager/base_tasks.py#L57-L153" signature="qiskit.passmanager.GenericPass" modifiers="class">

  A pass instance can read and write to the provided [`PropertySet`](qiskit.passmanager.PropertySet "qiskit.passmanager.PropertySet"), and may modify the input pass manager IR.
