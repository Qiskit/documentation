    The data points are returning in the form of a list of dictionaries:

    > *   **ydata\[i]\[‘mean’] is a list, where item**

    >
    > *   **ydata\[i]\[‘std’] is a list, where ydata\[‘std’]\[j] is the**
    >
    >     standard deviation of the success of qubit i.
