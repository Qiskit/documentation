# qiskit.pulse.builder.num\_qubits

<Function id="qiskit.pulse.builder.num_qubits" isDedicatedPage={true} github="https://github.com/qiskit/qiskit/tree/stable/0.21/qiskit/pulse/builder.py" signature="num_qubits()">
  Return number of qubits in the currently active backend.

  Examples:

  ```python
  from qiskit import pulse
  from qiskit.providers.fake_provider import FakeOpenPulse2Q

  backend = FakeOpenPulse2Q()

  with pulse.build(backend):
      print(pulse.num_qubits())
  ```

  ```
  2
  ```

  <Admonition title="Note" type="note">
    Requires the active builder context to have a backend set.
  </Admonition>

  **Return type**

  `int`
</Function>
