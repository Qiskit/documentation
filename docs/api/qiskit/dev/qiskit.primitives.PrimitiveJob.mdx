  Handle to a job from the reference implementations of the primitives in Qiskit.

  This is a concrete implementation of the [`BasePrimitiveJob`](qiskit.primitives.BasePrimitiveJob "qiskit.primitives.BasePrimitiveJob") interface. See the documentation of that class for a discussion of the interface.

  Primitives implementers looking to create their own job classes should not subclass this, but instead subclass the interface definition [`BasePrimitiveJob`](qiskit.primitives.BasePrimitiveJob "qiskit.primitives.BasePrimitiveJob").
