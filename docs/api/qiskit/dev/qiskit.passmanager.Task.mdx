# Task

<Class id="qiskit.passmanager.Task" isDedicatedPage={true} github="https://github.com/Qiskit/qiskit/tree/main/qiskit/passmanager/base_tasks.py#L31-L54" signature="qiskit.passmanager.Task" modifiers="class">

  An interface of the pass manager task.

  The task takes an IR, and outputs a (possibly different) IR after some operation on it. A task can rely on the [`PropertySet`](qiskit.passmanager.PropertySet "qiskit.passmanager.PropertySet") to communicate intermediate data among tasks.
