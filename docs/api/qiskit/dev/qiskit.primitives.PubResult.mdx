# PubResult

<Class id="qiskit.primitives.PubResult" isDedicatedPage={true} github="https://github.com/Qiskit/qiskit/tree/main/qiskit/primitives/containers/pub_result.py#L24-L65" signature="qiskit.primitives.PubResult(data, metadata=None)" modifiers="class">
  Bases: [`object`](https://docs.python.org/3/library/functions.html#object)

  The result object for a single pub (primitive unified bloc).

  Each [`PubResult`](#qiskit.primitives.PubResult "qiskit.primitives.PubResult") is a single element of a greater [`PrimitiveResult`](qiskit.primitives.PrimitiveResult "qiskit.primitives.PrimitiveResult"). Within this result, there is implementation-defined freeform [`metadata`](#qiskit.primitives.PubResult.metadata "qiskit.primitives.PubResult.metadata"), and a [`DataBin`](qiskit.primitives.DataBin "qiskit.primitives.DataBin") in the [`data`](#qiskit.primitives.PubResult.data "qiskit.primitives.PubResult.data") field.

  Most likely, you care about accessing the processed data of your execution. This is in the [`data`](#qiskit.primitives.PubResult.data "qiskit.primitives.PubResult.data") attribute. The [`metadata`](#qiskit.primitives.PubResult.metadata "qiskit.primitives.PubResult.metadata") object may contain extra information about the execution of this pub, including implementation-specific information, which should be documented by your primitive provider (as opposed to [`PrimitiveResult.metadata`](qiskit.primitives.PrimitiveResult#metadata "qiskit.primitives.PrimitiveResult.metadata"), which is metadata about the entire submission).

  You typically get instances of this class by iterating over or indexing into a [`PrimitiveResult`](qiskit.primitives.PrimitiveResult "qiskit.primitives.PrimitiveResult"), which is what you get from `MyPritimive().run().result()`.

  Initialize a pub result.

  **Parameters**

  *   **data** ([*DataBin*](qiskit.primitives.DataBin "qiskit.primitives.DataBin")) – Result data.
  *   **metadata** ([*dict*](https://docs.python.org/3/library/stdtypes.html#dict)*\[*[*str*](https://docs.python.org/3/library/stdtypes.html#str)*, Any] | None*) – Metadata specific to this pub. Keys are expected to be strings.
