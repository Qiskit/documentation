Functions for truncating Pauli operators within given error budgets.

| ------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------- |
| [`TruncationErrorBudget`](utils-truncating-truncation-error-budget "qiskit_addon_obp.utils.truncating.TruncationErrorBudget")  | A class for storing the constants that determine the truncation error budget. |
