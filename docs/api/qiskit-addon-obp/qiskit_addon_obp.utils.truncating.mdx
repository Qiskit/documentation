Functions for truncating Pauli operators within given error budgets.

|                                                                                                                                                 |                                                                               |
| ----------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| [`TruncationErrorBudget`](qiskit_addon_obp.utils.truncating.TruncationErrorBudget "qiskit_addon_obp.utils.truncating.TruncationErrorBudget")    | A class for storing the constants that determine the truncation error budget. |
| [`setup_budget`](qiskit_addon_obp.utils.truncating.setup_budget "qiskit_addon_obp.utils.truncating.setup_budget")                               | Calculate the budget available to each slice for observable term truncation.  |
| [`truncate_binary_search`](qiskit_addon_obp.utils.truncating.truncate_binary_search "qiskit_addon_obp.utils.truncating.truncate_binary_search") | Perform binary search to find an optimal observable truncation threshold.     |
