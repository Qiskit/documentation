# slice\_by\_barriers

<Function id="qiskit_addon_utils.slicing.slice_by_barriers" isDedicatedPage={true} github="https://github.com/Qiskit/qiskit-addon-utils/tree/main/qiskit_addon_utils/slicing/slice_by_barriers.py#L21-L52" signature="slice_by_barriers(circuit)">
  Split a `QuantumCircuit` into slices around full-circuit barriers.

  Barriers which do not act on all circuit qubits will be treated as normal operations and included in the slices. Barriers which act on all qubits will be interpreted as slice locations and will not be included in the output slices.

  **Parameters**

  **circuit** ([*QuantumCircuit*](/api/qiskit/qiskit.circuit.QuantumCircuit "(in Qiskit v1.2)")) â€“ The circuit to be split.

  **Returns**

  A sequence of [`QuantumCircuit`](/api/qiskit/qiskit.circuit.QuantumCircuit "(in Qiskit v1.2)") objects, one for each slice.
