  ### to\_dict

  <Function id="qiskit_ibm_runtime.noise_learner_v3.NoiseLearnerV3Results.to_dict" github="https://github.com/Qiskit/qiskit-ibm-runtime/tree/main/qiskit_ibm_runtime/noise_learner_v3/noise_learner_v3_result.py#L139-L186" signature="to_dict(instructions, require_refs=True)">
    Convert to a dictionary from `InjectNoise.ref` to `PauliLindbladMap` objects. This function iterates over a sequence of instructions, extracts the `ref` value from the inject noise annotation of each instruction, and returns a dictionary mapping those refs to the corresponding noise data (in `PauliLindbladMap` format) stored in this [`NoiseLearnerV3Results`](#qiskit_ibm_runtime.noise_learner_v3.NoiseLearnerV3Results "qiskit_ibm_runtime.noise_learner_v3.NoiseLearnerV3Results") object.

    **Parameters**

    *   **instructions** (*Sequence\[*[*CircuitInstruction*](/docs/api/qiskit/qiskit.circuit.CircuitInstruction)*]*) ŌĆō The instructions to get the refs from.
    *   **require\_refs** (*bool*) ŌĆō Whether to raise if some of the instructions do not own an inject noise annotation. If `False`, all the instructions that do not contain an inject noise annotations are simply skipped when constructing the returned dictionary.

    **Raises**

    *   **ValueError** ŌĆō If `instructions` contains a number of elements that is not equal to the item in this [`NoiseLearnerV3Results`](#qiskit_ibm_runtime.noise_learner_v3.NoiseLearnerV3Results "qiskit_ibm_runtime.noise_learner_v3.NoiseLearnerV3Results") object.
    *   **ValueError** ŌĆō If some of the instructions do not contain a box.
    *   **ValueError** ŌĆō If multiple instructions have the same `ref`.
    *   **ValueError** ŌĆō If some of the instructions have no inject noise annotation and `require_refs` if `True`.

    **Return type**

    dict\[int, [*PauliLindbladMap*](/docs/api/qiskit/qiskit.quantum_info.PauliLindbladMap)]
  </Function>
