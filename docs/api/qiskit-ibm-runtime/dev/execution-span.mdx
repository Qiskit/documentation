# Execution Spans

`qiskit_ibm_runtime.execution_span`

## Overview

An [`ExecutionSpans`](/docs/api/qiskit-ibm-runtime/dev/execution-span-execution-spans "qiskit_ibm_runtime.execution_span.ExecutionSpans") class instance is an iterable of [`ExecutionSpan`](/docs/api/qiskit-ibm-runtime/dev/execution-span-execution-span "qiskit_ibm_runtime.execution_span.ExecutionSpan")\s, where each iterand gives timing information about a chunk of data. Execution spans are returned as part of the metadata of a primitive job result.
