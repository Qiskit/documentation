# SamplexItemModel

<Class id="ibm_quantum_schemas.executor.version_1_1.SamplexItemModel" isDedicatedPage={true} github="https://github.com/Qiskit/ibm-quantum-schemas/tree/0.10rc2/ibm_quantum_schemas/executor/version_1_1/models.py#L126-L155" signature="SamplexItemModel" modifiers="pydantic model">
  Bases: `BaseModel`

  Execution specifications for a single quantum circuit.

  The circuit for each item is store separately in `QuantumProgramModel`.

  ### chunk\_size

  <Attribute id="ibm_quantum_schemas.executor.version_1_1.SamplexItemModel.chunk_size" attributeTypeHint="Annotated[int, Field(ge" attributeValue="1)] | Literal['auto'] = 'auto'" modifiers="field">
    The maximum number circuit arguments to bind to the circuit per shot loop.

    When `"auto"`, the number is chosen server-side with heuristics designed to optimize execution speed. A quantum program must have items where either all chunk sizes are integer-valued, or all chunk sizes are `"auto"`. Integer values are only allowed inside of session exection mode.
  </Attribute>

  ### shape

  <Attribute id="ibm_quantum_schemas.executor.version_1_1.SamplexItemModel.shape" attributeTypeHint="list[int] [Required]" modifiers="field">
