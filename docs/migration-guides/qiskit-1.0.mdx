# Qiskit 1.0 migration guide

Qiskit 1.0 brings new stability guarantees and performance improvements, but it
does break compatibility with all previous versions of Qiskit. See the
following guides on updating your environment and code to work with Qiskit v1.0.

- [Install the new Qiskit v1.0 package](/migration-guides/qiskit-1.0-installation) for upgrading Python environments and project dependencies.
- [Qiskit v1.0 feature changes](/migration-guides/qiskit-1.0-features) for updating your code.
- [Breaking package changes of Qiskit v1.0](/migration-guides/metapackage-migration) to understand the new and old package structure.

Read more about the benefits of Qiskit v1.0 on the [IBM&reg; blog](https://www.ibm.com/quantum/blog/qiskit-1-0-release-summary).
