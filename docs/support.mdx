# Support

### Get help with Qiskit and IBM Quantum Platform

| | |
| --- | --- |
| IBM Quantum&trade; Network members  | Reach out to IBM Quantum Support if you have technical difficulties, questions, or concerns. Ask your administrator or IBM&reg; representative for IBM Quantum Support contact information. |
| Pay-As-You-Go users  | Open a case in the IBM Cloud&reg; Support Center by following [this link](https://cloud.ibm.com/unifiedsupport/cases/form), and choosing 'Qiskit Runtime' from the Topic field. |
| Qiskit SDK users | Access the Qiskit Slack community: [Qiskit Slack](https://qisk.it/join-slack).  |
| Qiskit Runtime users | Join the `qiskit-runtime` channel within the [Qiskit Slack workspace](https://qisk.it/join-slack).|

### Discuss and ask questions

| | |
| --- | --- |
| Discussions on quantum information science and development with the larger quantum computing field | Visit [Quantum Computing Stack Exchange](https://quantumcomputing.stackexchange.com/questions/) site. Be sure to read the ["How do I ask a good question?"](https://quantumcomputing.stackexchange.com/help/how-to-ask) guide, and make use of tags such as `qiskit`, `ibm-runtime`, and `ibm-quantum-services` for best results.|
| Questions specific to programming |Visit [Stack Overflow](https://stackoverflow.com/) and use the tag `qiskit`.|


## Open-source governance

See the [Open-source resources](/open-source/) for details on contributing code to Qiskit. Information there includes the [code of conduct](/open-source/code-of-conduct), PDF license agreements for [individual](https://docs.quantum.ibm.com/open-source/qiskit-cla.pdf) and [corporate](https://docs.quantum.ibm.com/open-source/qiskit-corporate-cla.pdf) contributors, [version strategy](/open-source/qiskit-sdk-version-strategy), and the [contributor](https://github.com/Qiskit/qiskit/blob/main/CONTRIBUTING.md) and [maintainer](https://github.com/Qiskit/qiskit/blob/main/MAINTAINING.md) guides.

To report bugs, make feature requests, and see what's coming in future releases, visit the individual GitHub repositories:

[Qiskit SDK](https://github.com/Qiskit/qiskit)

[Qiskit Runtime](https://github.com/Qiskit/qiskit-ibm-runtime)

[Qiskit Serverless](https://github.com/Qiskit/qiskit-serverless)

[Qiskit Transpiler Service](https://github.com/Qiskit/qiskit-ibm-transpiler)

[Qiskit ecosystem](https://www.ibm.com/quantum/ecosystem) - Click on the Repo link for specific package to go to its GitHub repository
