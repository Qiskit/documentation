# Optimization Mapper Qiskit addon

[Qiskit addons](/docs/guides/addons) are a collection of modular tools for building utility-scale workloads powered by Qiskit.

This package contains functionality to model optimization problems and map them to operators. For example, this package contains functions for creating models of binary optimization problems to solve with Qiskit.

## Installation

We encourage installing this package via `pip`, when possible:

```bash
pip install 'qiskit-addon-opt-mapper'
```

For more installation information, refer to the [installation instructions](install) in the documentation.
