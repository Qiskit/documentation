# Qiskit addon: shaded lightcones (SLC)

[Qiskit addons](/docs/guides/addons) are a collection of modular tools for building utility-scale workloads powered by Qiskit.

This package contains the Qiskit addon for shaded lightcones (SLC). These can be used to reduce the sampling overhead of PEC.

## Documentation

All documentation is available [here](https://qiskit.github.io/qiskit-addon-slc/).

## Installation

We encourage installing this package via `pip`, when possible:

```bash
pip install qiskit-addon-slc
```

For more installation information refer to the [installation instructions](install) in the documentation.
