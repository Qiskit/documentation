# OpenQASM 3 feature table

Below is a list of the `OpenQASM 3` language features.

For more details on these capabilities, see the [OpenQASM 3.X Live Specification](https://openqasm.com/) .

Key:

*   ❌ Not supported
*   🟡 Partial support
*   ✅ Supported

| Feature                       | Qiskit SDK  |  Qiskit IBM Runtime  | Runtime Comments                                                                                                          |
| ------------------------------| ------------| ---------------------| ------------------------------------------------------------------------------------------------------------------------- |
| comments                      | ✅          |  ✅                  |                                                                                                                           |
| QASM version string           | ✅          |  ✅                  |                                                                                                                           |
| `include`                     | ✅          |  ✅                  |                                                                                                                           |
| unicode names                 | ✅          |  ✅                  |                                                                                                                           |
| `qubit`                       | ❌          |  ❌                  | Only supports physical qubit identifiers                                                                                 |
| `bit`                         | ✅          |  ✅                  |                                                                                                                           |
| `bool`                        | ✅          |  ✅                  |                                                                                                                           |
| `int`                         | ✅          |  ✅                  | Some support for comparisons against integers and casting                                                                |
| `uint`                        | 🟡          |  🟡                  | Support for compile time usage and casting                                                                                |
| `float`                       | 🟡          |  🟡                  | Support for compile time usage and casting                                                                                |
| `angle`                       | 🟡          |  🟡                  | Support for compile time usage and casting                                                                                |
| `complex`                     | ❌          |  ❌                  |                                                                                                                           |
| `const`                       | ❌          |  ❌                  |                                                                                                                           |
| `pi`/`π`/`tau`/`τ`/`euler`/`ℇ`| ✅          |  ✅                  |                                                                                                                           |
| Aliasing: let                 | ❌          |  ❌                  |                                                                                                                           |
| register concatenation        | ❌          |  ❌                  |                                                                                                                           |
| casting                       | 🟡          |  🟡                  | Casting between arrays of bit, int, and bool values is supported at run time. Casting of additional types is supported at compile time. |
| `duration`                    | ✅          |  ✅                  |                                                                                                                           |
| `durationof`                  | ❌          |  ❌                  |                                                                                                                           |
| `ns`/`µs`/`us`/`ms`/`s`/`dt`  | ✅          |  ✅                  |                                                                                                                           |
| `stretch`                     | ✅          |  🟡                  | Limited support for stretches.  See details in [Deferred timing resolution using stretch.](/docs/guides/stretch)                                                                                             |
| `delay`                       | ✅          |  ✅                  |                                                                                                                           |
| `barrier`                     | ✅          |  ✅                  |                                                                                                                           |
| `box`                         | ✅          |  ❌                  |                                                                                                                           |
| Built-in `U`                  | ✅          |  ✅                  |                                                                                                                           |
| `gate`                        | 🟡          |  🟡                  | No support for non-basis gates.                                                                                            |
| `gphase`                      | ❌          |  ❌                  |                                                                                                                           |
| `ctrl @`/ `negctrl @`         | ❌          |  ❌                  |                                                                                                                           |
| `inv @`                       | ❌          |  ❌                  |                                                                                                                           |
| `pow(k) @`                    | ❌          |  ❌                  |                                                                                                                           |
| `reset`                       | ✅          |  ✅                  |                                                                                                                           |
| `measure`                     | ✅          |  ✅                  |                                                                                                                           |
| bit operations                | ✅          |  ✅                  |                                                                                                                           |
| boolean operations            | ✅          |  ✅                  |                                                                                                                           |
| arithmetic expressions        | 🟡          |  🟡                  | Support for arithmetic expressions that can be computed at compile time                                                 |
| comparisons                   | ✅          |  ✅                  |                                                                                                                           |
| `if`                          | ✅          |  ✅                  |                                                                                                                           |
| `else`                        | ✅          |  ✅                  |                                                                                                                           |
| `else if`                     | ❌          |  ❌                  |                                                                                                                           |
| `for` loops                   | ❌          |  ❌                  |                                                                                                                           |
| `while` loops                 | ❌          |  ❌                  |                                                                                                                           |
| `continue`                    | ❌          |  ❌                  |                                                                                                                           |
| `break`                       | ❌          |  ❌                  |                                                                                                                           |
| `return`                      | ❌          |  ❌                  |                                                                                                                           |

| `def` subroutines (quantum)   | ❌          |  ❌                  |                                                                                                                           |
| `input`                       | 🟡          |  🟡                  |     Support for Rz angles only                                       |
| `output`                      | ❌          |  ❌                  |
