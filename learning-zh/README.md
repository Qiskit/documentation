## 使用原则

1. 保留 IBM 英文原文，不直接覆盖原始课程文件。
2. 中文内容以语义解释为主，不进行脱离上下文的逐句直译。
3. 数学公式、代码、变量名和 Qiskit API 保持原样。
4. 专业术语第一次出现时采用“中文（English）”格式。
5. 中文改编内容注明来源，并遵循 CC BY-SA 4.0 协议。
