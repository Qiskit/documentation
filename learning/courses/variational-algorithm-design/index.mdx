# Overview

!["Hero image"](/learning/images/courses/variational-algorithm-design/hero.svg)

This course is intended for individuals who have some experience with quantum computing and want to learn how to write variational algorithms using Qiskit Runtime primitives. It is also suitable for quantum computing professionals who want to expand their knowledge and skills in the field of variational algorithms.

## Exam

To earn your badge for Variational algorithm design, take the exam at IBM&reg; Training. This exam is intended to be taken after reading the lessons in this course. After you pass the exam, you will be notified by Credly that you have earned a badge.

[Exam](/learning/courses/variational-algorithm-design/exam)
