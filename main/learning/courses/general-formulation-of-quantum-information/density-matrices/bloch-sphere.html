Bloch sphere

There's a useful geometric way to represent qubit states known as the Bloch sphere. It's very convenient, but unfortunately it only works for qubits — the analogous representation no longer corresponds to a spherical object once we have three or more classical states of our system.

Qubit states as points on a sphere

Let's start by thinking about a quantum state vector of a qubit: α∣0⟩+β∣1⟩.\alpha \vert 0\rangle + \beta \vert 1\rangle.α∣0⟩+β∣1⟩. We can restrict our attention to vectors for which α\alphaα is a nonnegative real number because every qubit state vector is equivalent up to a global phase to one for which α≥0.\alpha \geq 0.α≥0. This allows us to write

∣ψ⟩=cos⁡(θ/2)∣0⟩+eiϕsin⁡(θ/2)∣1⟩\vert\psi\rangle = \cos\bigl(\theta/2\bigr) \vert 0\rangle + e^{i\phi} \sin\bigl(\theta/2\bigr) \vert 1\rangle∣ψ⟩=cos(θ/2)∣0⟩+eiϕsin(θ/2)∣1⟩

for two real numbers θ∈[0,π]\theta \in [0,\pi]θ∈[0,π] and ϕ∈[0,2π).\phi\in[0,2\pi).ϕ∈[0,2π). Here, we're allowing θ\thetaθ to range from 000 to π\piπ and dividing by 222 in the argument of sine and cosine because this is a conventional way to parameterize vectors of this sort, and it will make things simpler a bit later on.

Now, it isn't quite the case that the numbers θ\thetaθ and ϕ\phiϕ are uniquely determined by a given quantum state vector α∣0⟩+β∣1⟩,\alpha \vert 0\rangle + \beta \vert 1\rangle,α∣0⟩+β∣1⟩, but it is nearly so. In particular, if β=0,\beta = 0,β=0, then θ=0\theta = 0θ=0 and it doesn't make any difference what value ϕ\phiϕ takes, so it can be chosen arbitrarily. Similarly, if α=0,\alpha = 0,α=0, then θ=π,\theta = \pi,θ=π, and once again ϕ\phiϕ is irrelevant (as our state is equivalent to eiϕ∣1⟩e^{i\phi}\vert 1\rangleeiϕ∣1⟩ for any ϕ\phiϕ up to a global phase). If, however, neither α\alphaα nor β\betaβ is zero, then there's a unique choice for the pair (θ,ϕ)(\theta,\phi)(θ,ϕ) for which ∣ψ⟩\vert\psi\rangle∣ψ⟩ is equivalent to α∣0⟩+β∣1⟩\alpha\vert 0\rangle + \beta\vert 1\rangleα∣0⟩+β∣1⟩ up to a global phase.

Next, let's consider the density matrix representation of this state.

∣ψ⟩⟨ψ∣=(cos⁡2(θ/2)e−iϕcos⁡(θ/2)sin⁡(θ/2)eiϕcos⁡(θ/2)sin⁡(θ/2)sin⁡2(θ/2))\vert\psi\rangle\langle\psi\vert = \begin{pmatrix} \cos^2(\theta/2) & e^{-i\phi}\cos(\theta/2)\sin(\theta/2)\\[2mm] e^{i\phi}\cos(\theta/2)\sin(\theta/2) & \sin^2(\theta/2) \end{pmatrix}∣ψ⟩⟨ψ∣=(cos2(θ/2)eiϕcos(θ/2)sin(θ/2)​e−iϕcos(θ/2)sin(θ/2)sin2(θ/2)​)

We can use some trigonometric identities,

cos⁡2(θ/2)=1+cos⁡(θ)2,sin⁡2(θ/2)=1−cos⁡(θ)2,cos⁡(θ/2)sin⁡(θ/2)=sin⁡(θ)2,\begin{gathered} \cos^2(\theta/2) = \frac{1 + \cos(\theta)}{2},\\[2mm] \sin^2(\theta/2) = \frac{1 - \cos(\theta)}{2},\\[2mm] \cos(\theta/2) \sin(\theta/2) = \frac{\sin(\theta)}{2}, \end{gathered}cos2(θ/2)=21+cos(θ)​,sin2(θ/2)=21−cos(θ)​,cos(θ/2)sin(θ/2)=2sin(θ)​,​

as well as the formula eiϕ=cos⁡(ϕ)+isin⁡(ϕ),e^{i\phi} = \cos(\phi) + i\sin(\phi),eiϕ=cos(ϕ)+isin(ϕ), to simplify the density matrix as follows.

∣ψ⟩⟨ψ∣=12(1+cos⁡(θ)(cos⁡(ϕ)−isin⁡(ϕ))sin⁡(θ)(cos⁡(ϕ)+isin⁡(ϕ))sin⁡(θ)1−cos⁡(θ))\vert\psi\rangle\langle\psi\vert = \frac{1}{2} \begin{pmatrix} 1 + \cos(\theta) & (\cos(\phi) - i \sin(\phi)) \sin(\theta)\\[1mm] (\cos(\phi) + i \sin(\phi)) \sin(\theta) & 1 - \cos(\theta) \end{pmatrix}∣ψ⟩⟨ψ∣=21​(1+cos(θ)(cos(ϕ)+isin(ϕ))sin(θ)​(cos(ϕ)−isin(ϕ))sin(θ)1−cos(θ)​)

This makes it easy to express this density matrix as a linear combination of the Pauli matrices:

I=(1001),σx=(0110),σy=(0−ii0),σz=(100−1). \mathbb{I} = \begin{pmatrix} 1 & 0\\[1mm] 0 & 1 \end{pmatrix}, \quad \sigma_x = \begin{pmatrix} 0 & 1\\[1mm] 1 & 0 \end{pmatrix}, \quad \sigma_y = \begin{pmatrix} 0 & -i\\[1mm] i & 0 \end{pmatrix}, \quad \sigma_z = \begin{pmatrix} 1 & 0\\[1mm] 0 & -1 \end{pmatrix}.I=(10​01​),σx​=(01​10​),σy​=(0i​−i0​),σz​=(10​0−1​).

Specifically, we conclude that

∣ψ⟩⟨ψ∣=I+sin⁡(θ)cos⁡(ϕ)σx+sin⁡(θ)sin⁡(ϕ)σy+cos⁡(θ)σz2.\vert\psi\rangle\langle\psi\vert = \frac{\mathbb{I} + \sin(\theta) \cos(\phi)\sigma_x + \sin(\theta)\sin(\phi) \sigma_y + \cos(\theta) \sigma_z}{2}.∣ψ⟩⟨ψ∣=2I+sin(θ)cos(ϕ)σx​+sin(θ)sin(ϕ)σy​+cos(θ)σz​​.

The coefficients of σx,\sigma_x,σx​, σy,\sigma_y,σy​, and σz\sigma_zσz​ in the numerator of this expression are all real numbers, so we can collect them together to form a vector in an ordinary, three-dimensional Euclidean space.

(sin⁡(θ)cos⁡(ϕ),sin⁡(θ)sin⁡(ϕ),cos⁡(θ))\bigl(\sin(\theta) \cos(\phi), \sin(\theta)\sin(\phi), \cos(\theta)\bigr)(sin(θ)cos(ϕ),sin(θ)sin(ϕ),cos(θ))

In fact, this is a unit vector. Using spherical coordinates it can be written as (1,θ,ϕ).(1,\theta,\phi).(1,θ,ϕ). The first coordinate, 1,1,1, represents the radius or radial distance (which is always 111 in this case), θ\thetaθ represents the polar angle, and ϕ\phiϕ represents the azimuthal angle.

In words, thinking about a sphere as the planet Earth, the polar angle θ\thetaθ is how far we rotate south from the north pole to reach the point being described, from 000 to π=180∘,\pi = 180^{\circ},π=180∘, while the azimuthal angle ϕ\phiϕ is how far we rotate east from the prime meridian, from 000 to 2π=360∘.2\pi = 360^{\circ}.2π=360∘. This assumes that we define the prime meridian to be the curve on the surface of the sphere from one pole to the other that passes through the positive xxx-axis.

Every point on the sphere can be described in this way — which is to say that the points we obtain when we range over all possible pure states of a qubit correspond precisely to a sphere in 333 real dimensions. (This sphere is typically called the unit 222-sphere because the surface of this sphere is two-dimensional.)

When we associate points on the unit 222-sphere with pure states of qubits, we obtain the Bloch sphere representation these states.

Six important examples

  1. The standard basis {∣0⟩,∣1⟩}.\{\vert 0\rangle,\vert 1\rangle\}.{∣0⟩,∣1⟩}. Let's start with the state ∣0⟩.\vert 0\rangle.∣0⟩. As a density matrix it can be written like this.

    ∣0⟩⟨0∣=I+σz2\vert 0 \rangle \langle 0 \vert = \frac{\mathbb{I} + \sigma_z}{2}∣0⟩⟨0∣=2I+σz​​

    By collecting the coefficients of the Pauli matrices in the numerator, we see that the corresponding point on the unit 222-sphere using Cartesian coordinates is (0,0,1).(0,0,1).(0,0,1). In spherical coordinates this point is (1,0,ϕ),(1,0,\phi),(1,0,ϕ), where ϕ\phiϕ can be any angle. This is consistent with the expression

    ∣0⟩=cos⁡(0)∣0⟩+eiϕsin⁡(0)∣1⟩,\vert 0\rangle = \cos(0) \vert 0\rangle + e^{i \phi} \sin(0) \vert 1\rangle,∣0⟩=cos(0)∣0⟩+eiϕsin(0)∣1⟩,

    which also works for any ϕ.\phi.ϕ. Intuitively speaking, the polar angle θ\thetaθ is zero, so we're at the north pole of the Bloch sphere, where the azimuthal angle is irrelevant.

    Along similar lines, the density matrix for the state ∣1⟩\vert 1\rangle∣1⟩ can be written like so.

    ∣1⟩⟨1∣=I−σz2\vert 1 \rangle \langle 1 \vert = \frac{\mathbb{I} - \sigma_z}{2}∣1⟩⟨1∣=2I−σz​​

    This time the Cartesian coordinates are (0,0,−1).(0,0,-1).(0,0,−1). In spherical coordinates this point is (1,π,ϕ)(1,\pi,\phi)(1,π,ϕ) where ϕ\phiϕ can be any angle. In this case the polar angle is all the way to π,\pi,π, so we're at the south pole where the azimuthal angle is again irrelevant.

  2. The basis {∣+⟩,∣−⟩}.\{\vert + \rangle, \vert - \rangle\}.{∣+⟩,∣−⟩}. We have these expressions for the density matrices corresponding to these states.

    ∣+⟩⟨+∣=I+σx2∣−⟩⟨−∣=I−σx2\begin{aligned} \vert {+} \rangle\langle {+} \vert & = \frac{\mathbb{I} + \sigma_x}{2}\\[2mm] \vert {-} \rangle\langle {-} \vert & = \frac{\mathbb{I} - \sigma_x}{2} \end{aligned}∣+⟩⟨+∣∣−⟩⟨−∣​=2I+σx​​=2I−σx​​​

    The corresponding points on the unit 222-sphere have Cartesian coordinates (1,0,0)(1,0,0)(1,0,0) and (−1,0,0),(-1,0,0),(−1,0,0), and spherical coordinates (1,π/2,0)(1,\pi/2,0)(1,π/2,0) and (1,π/2,π),(1,\pi/2,\pi),(1,π/2,π), respectively.

    In words, ∣+⟩\vert +\rangle∣+⟩ corresponds to the point where the positive xxx-axis intersects the unit 222-sphere and ∣−⟩\vert -\rangle∣−⟩ corresponds to the point where the negative xxx-axis intersects it. More intuitively, ∣+⟩\vert +\rangle∣+⟩ is on the equator of the Bloch sphere where it meets the prime meridian, and ∣−⟩\vert - \rangle∣−⟩ is on the equator on the opposite side of the sphere.

  3. The basis {∣+i⟩,∣−i⟩}.\{\vert {+i} \rangle, \vert {-i} \rangle\}.{∣+i⟩,∣−i⟩}. As we saw earlier in the lesson, these two states are defined like this:

    ∣+i⟩=12∣0⟩+i2∣1⟩∣−i⟩=12∣0⟩−i2∣1⟩.\begin{aligned} \vert {+i} \rangle & = \frac{1}{\sqrt{2}} \vert 0 \rangle + \frac{i}{\sqrt{2}} \vert 1 \rangle\\[2mm] \vert {-i} \rangle & = \frac{1}{\sqrt{2}} \vert 0 \rangle - \frac{i}{\sqrt{2}} \vert 1 \rangle. \end{aligned}∣+i⟩∣−i⟩​=2​1​∣0⟩+2​i​∣1⟩=2​1​∣0⟩−2​i​∣1⟩.​

    This time we have these expressions.

    ∣+i⟩⟨+i∣=I+σy2∣−i⟩⟨−i∣=I−σy2\begin{aligned} \vert {+i} \rangle\langle {+i} \vert & = \frac{\mathbb{I} + \sigma_y}{2}\\[2mm] \vert {-i} \rangle\langle {-i} \vert & = \frac{\mathbb{I} - \sigma_y}{2} \end{aligned}∣+i⟩⟨+i∣∣−i⟩⟨−i∣​=2I+σy​​=2I−σy​​​

    The corresponding points on the unit 222-sphere have Cartesian coordinates (0,1,0)(0,1,0)(0,1,0) and (0,−1,0),(0,-1,0),(0,−1,0), and spherical coordinates (1,π/2,π/2)(1,\pi/2,\pi/2)(1,π/2,π/2) and (1,π/2,3π/2),(1,\pi/2,3\pi/2),(1,π/2,3π/2), respectively.

    In words, ∣+i⟩\vert {+i} \rangle∣+i⟩ corresponds to the point where the positive yyy-axis intersects the unit 222-sphere and ∣−i⟩\vert {-i} \rangle∣−i⟩ to the point where the negative yyy-axis intersects it.

∣ψα⟩=cos⁡(α)∣0⟩+sin⁡(α)∣1⟩(for α∈[0,π))\vert \psi_{\alpha} \rangle = \cos(\alpha) \vert 0\rangle + \sin(\alpha) \vert 1\rangle \qquad \text{(for $\alpha \in [0,\pi)$)}∣ψα​⟩=cos(α)∣0⟩+sin(α)∣1⟩(for α∈[0,π))

Convex combinations of points

Similar to what we have already discussed for density matrices, we can take convex combinations of points on the Bloch sphere to obtain representations of qubit density matrices. In general, this results in points inside of the Bloch sphere, which represent density matrices of states that are not pure. Sometimes we refer to the Bloch ball when we wish to be explicit about the inclusion of points inside of the Bloch sphere as representations of qubit density matrices.

For example, we've seen that the density matrix 12I,\frac{1}{2}\mathbb{I},21​I, which represents the completely mixed state of a qubit, can be written in these two alternative ways:

12I=12∣0⟩⟨0∣+12∣1⟩⟨1∣and12I=12∣+⟩⟨+∣+12∣−⟩⟨−∣.\frac{1}{2} \mathbb{I} = \frac{1}{2} \vert 0\rangle\langle 0\vert + \frac{1}{2} \vert 1\rangle\langle 1\vert \quad\text{and}\quad \frac{1}{2} \mathbb{I} = \frac{1}{2} \vert +\rangle\langle +\vert + \frac{1}{2} \vert -\rangle\langle -\vert.21​I=21​∣0⟩⟨0∣+21​∣1⟩⟨1∣and21​I=21​∣+⟩⟨+∣+21​∣−⟩⟨−∣.

We also have

12I=12∣+i⟩⟨+i∣+12∣−i⟩⟨−i∣,\frac{1}{2} \mathbb{I} = \frac{1}{2} \vert {+i}\rangle\langle {+i} \vert + \frac{1}{2} \vert {-i} \rangle\langle {-i}\vert,21​I=21​∣+i⟩⟨+i∣+21​∣−i⟩⟨−i∣,

and more generally we can use any two orthogonal qubit state vectors (which will always correspond to two antipodal points on the Bloch sphere). If we average the corresponding points on the Bloch sphere in a similar way, we obtain the same point, which in this case is at the center of the sphere. This is consistent with the observation that

12I=I+0⋅σx+0⋅σy+0⋅σz2,\frac{1}{2} \mathbb{I} = \frac{\mathbb{I} + 0 \cdot \sigma_x + 0 \cdot \sigma_y + 0 \cdot \sigma_z}{2},21​I=2I+0⋅σx​+0⋅σy​+0⋅σz​​,

giving us the Cartesian coordinates (0,0,0).(0,0,0).(0,0,0).
